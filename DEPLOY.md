# Інструкція зі встановлення Node.js та Gulp.js на Windows 10

## 0. Git Hub

В залежності від задачі, завантажуй або повністю те що я тобі вислав, або тільки папку build 

## 1. Перевірка наявних інсталяцій

### Перевірка Node.js
1. Відкрийте **Командний рядок** (Command Prompt) або **PowerShell**:
   - Натисніть `Win + R`, введіть `cmd` або `powershell`, натисніть Enter

2. Перевірте, чи встановлений Node.js:
   ```cmd
   node --version
   ```

3. Перевірте, чи встановлений менеджер пакетів npm (йде разом з Node.js):
   ```cmd
   npm --version
   ```

4. Якщо команди не розпізнаються або показують помилку — Node.js не встановлений або не доданий до змінних середовища PATH.

### Перевірка Gulp.js
1. У тому ж терміналі перевірте наявність глобально встановленого Gulp:
   ```cmd
   gulp --version
   ```

2. Якщо команда не знайдена — Gulp не встановлений глобально.

## 2. Встановлення Node.js

### Крок 1: Завантаження інсталятора
1. Перейдіть на офіційний сайт: [https://nodejs.org/](https://nodejs.org/)
2. Завантажте **LTS версію** (Recommended For Most Users)
3. Збережіть файл `.msi` на вашому комп'ютері

### Крок 2: Встановлення
1. Запустіть завантажений інсталятор
2. Дотримуйтесь вказівок майстра встановлення:
   - Прийміть ліцензійну угоду
   - Оберіть місце встановлення (рекомендується залишити за замовчуванням)
   - **Важливо**: На екрані "Custom Setup" оберіть опцію **"Add to PATH"**
   - Натисніть "Install"

### Крок 3: Перевірка встановлення
1. Закрийте та відкрийте заново термінал (Command Prompt/PowerShell)
2. Виконайте команди для перевірки:
   ```cmd
   node --version
   npm --version
   ```
3. Якщо відображаються версії — Node.js успішно встановлено.

## 3. Встановлення Gulp.js

### Крок 1: Глобальне встановлення Gulp CLI (Command Line Interface)
1. У терміналі виконайте команду:
   ```cmd
   npm install --global gulp-cli
   ```

2. Перевірте встановлення:
   ```cmd
   gulp --version
   ```
   Має відобразитись версія Gulp CLI (наприклад, `CLI version: 2.3.0`).

### Крок 2: Локальне встановлення Gulp у проекті
1. Створіть або перейдіть до папки вашого проекту:
   ```cmd
   cd шлях/до/вашого/проекту
   ```

2. Ініціалізуйте npm у проекті (якщо ще не зроблено):
   ```cmd
   npm init -y
   ```

## 4. Перевірка повної установки

### Фінальна перевірка всіх компонентів:
```cmd
node --version
npm --version
gulp --version
```

Очікуваний результат:
- Версія Node.js (наприклад, `v18.15.0`)
- Версія npm (наприклад, `9.5.0`)
- Версія Gulp CLI (наприклад, `CLI version: 2.3.0`)

## 5. Розгортання проєкту

### Коротко про структуру
- `gulpfile.js` — таски Gulp.
- `src/` — вихідні файли (стилі, скрипти, шаблони).
- `build/` — результати білду.

1. встановити залежності
   ```cmd
   npm install
   ```
2. запустити режим розробки (локальний сервер, слідкування за файлами)
   ```cmd
   npm run dev
   ```
3. якщо потрібно зібрати продакшен-білд
   ```cmd
   npm run build
   ```
